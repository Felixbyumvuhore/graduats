Mobile Menu Test Page

Resize your browser to mobile size (< 768px) and click the hamburger menu (3 lines) in the top right.


Instructions:

  1. Open browser developer tools (F12)
  2. Toggle device toolbar (Ctrl+Shift+M)
  3. Select a mobile device or resize to < 768px
  4. Click the hamburger icon (3 horizontal lines)
  5. Menu should slide in from the left

What Should Happen:

  • ✅ Hamburger icon appears on mobile
  • ✅ Click hamburger → menu slides in from left
  • ✅ Hamburger transforms to X icon
  • ✅ Dark backdrop appears behind menu
  • ✅ Click backdrop or link → menu closes
